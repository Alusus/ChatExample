# مثال الدردشة

[[English]](readme.md)

مثال على تطبيق دردشة ويب باستخدام لغة الأسس ومنصة الويب. يتوفر هذا المثال باللهجتين العربية والإنجليزية
من لغة الأسس البرمجية.

## الإعداد

1. ثبت
  [لغة الأسس](https://alusus.org/ar/download.html).

2. ثبت Docker.

3. شغل قاعدة البيانات:

<div dir=ltr>

  ```
  docker compose up -d
  ```

</div>

4. شغل التطبيق:
  ```
  الأسس بداية.أسس ابدأ
  ```
  
 <div dir=ltr>
 
  ```
  alusus main.alusus start
  ```

</div>

## نشر التطبيق على منصة الأسس نت

لنشر التطبيق، قم بتشغيل الأمر التالي واتبع التعليمات:

```
الأسس بداية.أسس انشر
```

<div dir=ltr>

```
alusus main.alusus publish
```

</div>

</div>
